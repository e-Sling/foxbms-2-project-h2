How to update the foxBMS application using the bootloader
=========================================================

1. Connect the foxBMS 2 hardware via a CAN-USB converter to the host computer
1. Make sure no other component except the BMS is active on the CAN bus
1. Start the bootloader update process on the host computer
1. Power-cycle the BMS
1. The update program should now be executed.
1. Power-cycle the BMS
